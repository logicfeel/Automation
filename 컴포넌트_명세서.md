# 컴포넌트 명세서 

| 컴포넌트  | 명칭      | 컴포넌트 종류 | 설명        | 비고    | 
|:-------:|:---------|:----------|:-----------|:-------|
| **SYS** | system   | M         | 시스템      | - | 
| **CMN** | common   | V, C      | 공통        | - | 
| **ADM** | admin    | M, V, C   | 관리자      | - | 

> - 컴포넌트 종류는 다를수 있다.
>   - M:모델 전용, V:뷰전용, VC:뷰컨트롤 전용
> - ...

> => 전역 값은 같으나
> system 은 전체의 종속성이 강하며, common 은 선택적 종속이다.
>   - 권한관리 : SYS
>   - 에러로그 : *SYS, CMN
>   - NPM : SYS
>   - 코드명령규칙 : SYS
>   - DB 접근 : CMN
